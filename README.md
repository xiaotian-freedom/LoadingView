# LoadingView
可以实现以下功能：

1、整个view的大小。

2、自定义圆圈的颜色、边框的大小、转动的速度。

3、加载成功及失败时的颜色、动画时长。 

4、加载成功和失败可以选择是否回调动画结束接口。 

5、自定义圆圈、成功、失败的动画路径。 

6、重置加载动画。
